## 🤖 Machine Learning Features

StudyWise-AI uses machine learning and NLP techniques to assist students with:

* Question analysis

* Text summarization

* Concept explanation

* Keyword extraction

* Personalized study support

### Technologies Used

* Python

* Scikit-learn

* Pandas

* NumPy

* NLP (Natural Language Processing)

### Objective

The ML module helps students understand topics faster by automatically processing educational content and generating useful study assistance.
